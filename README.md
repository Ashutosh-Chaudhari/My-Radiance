# My Radiance - Holistic Skincare Awareness Platform

A user-centric web application designed to provide structured, science-backed skincare guidance. This project addresses the common issues of information fragmentation and commercial bias found in traditional skincare platforms.

## üåü The "Why" behind the Project
Existing skincare platforms often over-focus on sales and lack mobile accessibility. My Radiance was built to provide:
1. **Science-Based Education:** Dedicated modules for understanding skin biology and the five common skin types (Oily, Dry, Combination, Normal, Sensitive).
2. **Structured Remedies:** Data-categorized home solutions organized by skin concern to ensure user clarity.
3. **Professional Accessibility:** Integrated pathways for users to seek guidance from certified dermatologists.
